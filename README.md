### keymaps basics

- keymaps based on {h,j,k,l}
- mod + {direction}  = focus
- mod + shift + {direction} = move
- mod + ctrl + {direction} = windowsizing
- mod + alt + (shift +) {direction} = focus/(move to) monitor
